# Object Types

### For Example:

    let studentName = {
    name: "Akbar Ali",
    age: 17
    }

    // Getting the Output
    console.log(studentName);
    console.log(studentName.name);
    console.log(studentName["age"]);

### Output
    { name: 'Akbar Ali', age: 17 }
    Akbar Ali
    17
